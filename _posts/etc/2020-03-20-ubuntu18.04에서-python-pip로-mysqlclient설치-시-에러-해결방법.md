Ubuntu18.04 환경에서  
pip install mysqlclient 에러

## 1. install ```libmysqlclient-dev```, ```gcc```
```bash
$ sudo apt install 
```

## 2. 다시 pip install
```bash
$ pip install mysqlclient
```
